> 제가 C++ 프로그래밍에 대해 C도 모르면서 구현을 막 해보던 것이라 에러가 있을 수도 있고,
> 객체화가 안되어있기에 C++ 이라고 표현하기 어렵지만, 함수 오버로딩과 입출력문이 간편하여 사용하게 되었고
